# Deployment Artifacts

This directory contains deployment information for each network.

## Files

- `localhost.json` - Local Anvil deployment
- `sepolia.json` - Sepolia testnet deployment
- `mainnet.json` - Ethereum mainnet deployment (production)

Each file contains:
- Network information
- Contract addresses
- Deployment configuration
- Deployer address
- Timestamp

## Usage

These files are automatically generated by the deployment script.

Do not manually edit these files.
